IEEE MASS 2026 Workshop

7th Network Meets Intelligent Computations (NMIC 2026)

About the Workshop

The NMIC 2026 workshop solicits papers addressing distributed computations for networking, intelligent computations supported by novel networking technologies, and computation-network convergence.

Call for Papers

Topics include:
  • AI for networking
  • Distributed machine learning
  • Edge computing
  • Blockchain and networking
  • 6G and future networks
  • AI Agents and LLM applications
  • Network security and privacy
  • IoT and smart city applications

The workshop explores convergence between intelligent computation technologies and emerging networking paradigms.
